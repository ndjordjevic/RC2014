RC2014 Micro — Glossary

Canonical vocabulary for reading the RC2014 Micro schematic (Rev 1.1, 2019). Introduced in Lesson 01.

Reading a schematic

Reference designator
The letter+number label that names a part on the board: U=integrated circuit, R=resistor, C=capacitor, Y=crystal, SW=switch, S/JP=jumper, Con/J=connector. E.g. U4 is the Z80.
Avoid: "part number" (that's the chip type, e.g. 27C512)
Net
One electrical connection — a wire and everything tied to it. Two pins on the same net are the same node.
Net label / off-page connector
A named tag on a wire (e.g. CLK, RESET, MREQ). Every wire with the same label is the same net even with no line drawn between them — this is how a dense schematic avoids spaghetti.
Avoid: "signal name" used loosely — here a label literally is the connection
Bus
A group of related nets carrying one logical thing, drawn as a thick line. The Micro has three: the address bus (A0–A15), the data bus (D0–D7), and the control bus (RD, WR, MREQ, …).
Power symbol
The supply tags: VCC (+5 V) and GND (0 V). They appear all over the sheet; every VCC tag is the same +5 V net.
Active-low (overbar)
A signal that is "true / asserted" when it is at 0 V (low), shown with a bar over the name (e.g. RD, CE, WE). Written in text as /RD or RD#.
Avoid: assuming high = active

The chips & parts (this board)

Z80 (CPU)
The 8-bit processor (U4); drives the address bus, exchanges bytes on the data bus, and emits control signals. 40-pin.
SRAM
Static RAM — read/write memory that holds data while powered, no refresh needed. The Micro's RAM is the HM62256 (U3), 32K×8.
EPROM
Erasable programmable ROM — non-volatile memory holding firmware. The Micro's ROM is the 27C512 (U6), 64K×8. Variant: the Micro+ uses a 39SF010 (128K) flash instead.
ACIA
Asynchronous Communications Interface Adapter — a serial UART chip. The Micro uses the MC68B50 (U2) for the console serial port.
Decoupling capacitor
A small cap across VCC–GND beside each chip (C1–C5) that supplies instant current and quiets supply noise.
Crystal oscillator
A quartz crystal (Y1, 7.3728 MHz) plus inverter(s) and caps that generate the steady CLK the CPU runs on.

Z80 signals you'll meet

CLK
The clock that paces every CPU action.
RESET
Forces the CPU back to a known start state (begins executing at address 0x0000).
M1
Marks an opcode-fetch machine cycle (the first byte of an instruction).
MREQ / IORQ
The CPU is addressing memory (MREQ) vs an I/O device (IORQ). The Z80 has two separate address spaces.
RD / WR
Direction of the current cycle: read from, or write to, the addressed location.
CE/CS, OE, WE
Chip Enable/Select, Output Enable, Write Enable — how a memory chip is told "you're the one being talked to," and whether to drive or accept data.
Bank selection
Using high address lines (here ROM A13/A14/A15, set by jumpers JP2/JP3/JP4 via the select nets S1/S2/S3) to choose which 8K slice of a larger ROM is visible. See the factory ROM 8-digit label scheme.
