**Index entry.** RC2014Z80 is the official GitHub organisation for the RC2014 — the code/hardware home behind the [[rc2014.co.uk]] shop. This page indexes the org's repositories and a few notable resources within them; we can deep-dive into any individual repo, the z88dk guide, or the ROM set later.

_All claims below are sourced from ../../raw/web/rc2014z80.md unless otherwise noted._

## Repositories
- **RC2014** (357★, Assembly) — flagship: hardware schematics (`Hardware/`), ROM build trees (`ROMs/`: CPM-IDE, Factory, MSBASIC, SIO2, TinyBasicPlus, pigfx, snake), CP/M disk/CF images (`CPM/`), and example BASIC programs (`BASIC-Programs/`). Latest release cpm-ide-v2.4 (2025-06). Third-party projects are git submodules (`git submodule update --init`).
- **picoterm** (95★, C) — Raspberry Pi Pico VGA terminal emulator for the RC2014.
- **RC2014-BASIC-Programs** (32★, BASIC) — runnable MS-BASIC programs.
- **z88dk** (12★, C) — org mirror of the Z80 development kit (C compiler, assembler, linker, libraries).
- **Bubble-display-toolbox** (4★, Assembly) — example code for the Bubble Display module.

## Notable resources
- **Using Z88DK with RC2014** (RC2014 repo wiki) — how to cross-compile C with z88dk: build from source, the two compilers (sccz80, zsdcc/SDCC), and the six `+rc2014` subtypes (ACIA, SIO, BASIC@0x9000, CPM@0x0100, HBIOS/RomWBW@0x0100, 8085). Example: `zcc +rc2014 -subtype=basic -SO3 ... -create-app`.
- **Factory ROMs** (`RC2014/ROMs/Factory`) — the official factory ROM images plus the 8-digit ROM-label decoder: each digit names the firmware in one 8K bank (R/K = MS-BASIC, 1/6 = CP/M Monitor, 88/9 = Small Computer Monitor, 0 = free), selected by the A13/A14/A15 jumpers; "plus" files are 128K for Micro Plus / Classic II Plus.
