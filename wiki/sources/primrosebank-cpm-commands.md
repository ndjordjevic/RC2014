## What it covers
- The CP/M Console Command Processor (CCP): on boot it shows the logged-on drive prompt (e.g. `A>`) and waits for input.
- The distinction between **internal** commands (built into the CCP) and **transient** commands (external programs / applications loaded and run).
- A brief overview of the CCP commands (part of a broader CP/M reference set: About, Commands, Cookbook, Emulation, Structure).
