## Landing page — CP/M Commands (primrosebank.net)

Part of primrosebank.net's Memotech MTX / CP/M reference section (sibling pages: CP/M About, Commands, Cookbook, Emulation, Structure). The page explains the CP/M Console Command Processor (CCP): once booted, the CCP displays the logged-on drive prompt (typically `A>`) and a cursor, then acts on user input by executing one of a limited number of **internal** commands or by calling an external program (a **transient** command; application programs are also transient commands). The page gives a brief overview of the CCP commands, noting fuller descriptions are widely available elsewhere. A practical CP/M command reference for users of Z80 CP/M machines like the RC2014.
