## Landing page — Small Computer Monitor v1.0

The Small Computer Monitor (SCM) is "a classic machine code monitor for Z80 based systems," by Steve Cousins / Small Computer Central. Version 1.0 is the current stable release; v1.1 is available for newer hardware.

Features: machine-code monitoring and debugging on Z80 (inline assembler/disassembler, register and memory examine/set, breakpoints, single-step, Intel .hex loading). Known limitation: single-stepping doesn't execute instructions where the program counter is unchanged (e.g. non-branching conditional jumps).

Supported hardware: RC2014 official systems; the Small Computer series (SC101, SC108, SC111, SC114, SC118); LiNC80; Bill Shen's Z280RC and Z80SBC64; Tom Szolyga's SBC version C; Z80 Small Computer (Z80sc).

Documentation/downloads (edition 1.0.0): Installation guide, Reference Sheet, Tutorial, User Guide, plus an Italian tutorial translation. The SCWorkshop v0.2.0 package includes source code, pre-built ROM images, and documentation. Older versions (0.1.9, 0.1.7, 0.1.6) are archived.
