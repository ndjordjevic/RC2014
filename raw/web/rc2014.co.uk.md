# RC2014 - Homebrew Z80 Computer

## Hero Description
"RC2014 is a simple 8 bit Z80 based modular computer originally built to run Microsoft BASIC. It is inspired by the home built computers of the late 70s and computer revolution of the early 80s."

The system features 8K ROM, 32K RAM, runs at 7.3728MHz, and communicates over serial at 115,200 baud.

## Full Kit page — RC2014 Micro — https://rc2014.co.uk/full-kits/rc2014-micro/

The RC2014 Micro is "a 32k Z80 based single board computer which runs either Microsoft BASIC or Steve Cousins SCM monitor ROM." It is a stripped-down RC2014 Mini, dropping the Pi Zero connector, power socket, LED, and keyboard connector to reduce cost and PCB size while keeping the standard RC2014 module form factor.

Core components: Z80 CPU, 27C512 EPROM, 62256 RAM (32KB), MC68B50 UART, 7.3728 MHz crystal. Serial via FTDI cable at 115200 baud (8-N-1). Supporting logic: 74HCT04 and 74HCT32, with passives (1kΩ×4, 1MΩ, 2.2kΩ, 3.3kΩ; 22pF×2, 100nF×6) and a tactile reset switch. Kit = one PCB plus all components, 3-pin and 6-pin right-angle headers, and a jumper. With a 40-way right-angle pin header it connects to a backplane (Backplane 5, 8, or Pro) for modules (Digital I/O, ESP8266 WiFi, sound cards, prototype boards). Sold via z80kits.com. Docs: assembly guide, schematic, and SCM tutorial PDFs.

## Article — BASIC, CP/M, RomWBW or Small Computer Monitor — https://rc2014.co.uk/1716/basic-cp-m-romwbw-or-small-computer-monitor/

The RC2014 offers four software/firmware environments:

- **Microsoft BASIC 4.7** — derived from NASCOM 2 BASIC, modified by Grant Searle (hardware-specific code stripped for a more generic BASIC, extra commands added). 32k version (Mini/Classic/Plus/Pro; ~56,958 free bytes) and 56k version (Pro or expanded Classic/Plus; ~32,382 free bytes). Boots to a memory prompt then the interpreter.
- **CP/M 2.2** — adapted by Grant Searle; native on the Pro or via upgrade kit on Classic/Plus. ROM-based Monitor supporting pageable ROM and either 68B50 ACIA or SIO/2 UART. Loads from Compact Flash; advantage over ROM BASIC is the ability to load and save programs from the CF card. Boot: "Press [SPACE] to activate console" then a monitor menu (ROM BASIC / CP/M / Intel-Hex loading).
- **RomWBW** — for models with the 512k ROM / 512k RAM upgrade; a ROM-based implementation of CP/M-80 2.2 and Z-System, supporting floppy, hard disk (IDE/CF/SD), video, and keyboard with VT-100 terminal emulation. Prebuilt ROM images plus source and build tools.
- **Small Computer Monitor (SCM)** — by Steve Cousins; a Z80 machine-code monitor with inline assembler/disassembler, register examine/set, breakpoints, single-step, and Intel .hex loading. Version "9" (8k, any model) and "88" (16k, Pro, with BASIC and CP/M loaders). 32k–64k RAM, both serial types.
