## New Product Launch - Pi Pico VGA Terminal  
  
|  |  ![](RC2014%20Newsletter%20-%20Issue%2039_files/dde40d94-2e90-923c-e8fe-bfd34a417172.jpg)  
---  
Last week the [Pi Pico VGA Terminal](https://rc2014.co.uk/modules/pi-pico-vga-terminal/) was launched. This module uses a Raspberry Pi Pico running PicoTerm to deliver either 80 column mono or 40 column colour text to a VGA monitor. VT100 style escape codes are supported, as well as some USB keyboards.  
  
This is available exclusively at [z80kits.com](https://z80kits.com/shop/rc2014-pi-pico-vga-terminal/)
