## CPNET-Z80 with Wiznet Ethernet Module

Jay Cotton has been working on a RC2014 compatible version of DRI CPNET to run on his MT011 networkd module. See the [CPNET-Z80](https://github.com/durgadas311/cpnet-z80) GitHub repo and Marks [MT011 Network Module](https://github.com/markt4311/MT011), or follow along with the discussion on the [RC2014 Google Group](https://groups.google.com/forum/#!msg/rc2014-z80/T7j-9CAsKAo/pLuXJ34mBwAJ).

## AY-3-8912 Adapter for RC2014 Sound Card

The [RC2014 Sound Card](https://www.tindie.com/products/semachthemonkey/ym2149-sound-card-for-rc2014-retro-computer/) has been popular, but not everybody has a spare YM2149 or YM-3-8910 for it. If you have a spare AY-3-8912 lying around, then this [Adapter PCB](https://www.tindie.com/products/semachthemonkey/adapter-pcb-for-ay-3-8912-sound-chip-to-ay-3-8910/) is just what you need!

## VT132 Module

The VT132 Module mentioned in the last newsletter is due for release next month. This DEC terminal emulator, VGA adapter, PS/2 keyboard adapter, Wifi Modem, etc, etc, looks great and is in beta testing at the moment. There's a link on [The High Nibble](https://thehighnibble.com/vt132/) website to be notified when it is live
