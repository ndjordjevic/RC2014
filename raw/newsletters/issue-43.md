## New Product - Micro SD Card Module

Last month saw the launch of the [Micro SD Card Module](https://z80kits.com/shop/micro-sd-card-module/) by [Extreme Kits](https://github.com/ExtremeElectronics/Pico-SD--IDE-interface--for-Rc2014). This is a drop in replacement for the compact flash module supplied with the RC2014 Classic II CP/M kit, the RC2014 Pro or the RC2014 Zed models, and requires no software changes on the RC2014 machine. This clever device uses a Pi Pico to emulate a compact flash card, and serves up the CF image file from the SD   
|  |  ![](RC2014%20Newsletter%20-%20Issue%2043_files/5c6dd3c4-f8dc-b218-2820-dfe48430a009.jpg)  
---  
card to either regular CP/M 2.2 RC2014 or a RomWBW based RC2014.  
  
There is even a Fast File Serial (FFS!) application that transfers files to or from a host computer super quickly. Read more about it on [GitHub](https://github.com/ExtremeElectronics/RC20XX-file-transfer-programs)

## Amstrad CPC BASIC 

  
Not wanting to choose between his RC2014 and Amstrad CPC, Mike Sutton, aka [Bread80](https://bread80.com/), ported BASIC to the RC2014. Available on [GitHub](https://github.com/Bread80/CPCForRC2014) now.

## CP/M IDE

  
If you like your file systems FAT then [Feilipu](https://feilipu.me/2022/03/23/cpm-ide-for-rc2014/) has something for you. [CP/M IDE](https://github.com/RC2014Z80/RC2014) has finally been released! This works on a standard RC2014 Pro, or other models with an [IDE Hard Drive Module](https://z80kits.com/shop/rc2014-ide-hard-drive-module/) (more stock due next month)
