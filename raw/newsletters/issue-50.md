## USB Storage for RomWBW based RC2014s!

In January 1996 the USB standard was launched. And in January 2024 a [USB storage module](<https://z80kits.com/shop/ch375-usb-storage-module/>) comes to the [RC2014 Zed](<https://z80kits.com/shop/rc2014-zed/>) and [Zed Pro](<https://z80kits.com/shop/rc2014-zed-pro/>)! This CH375 based adapter allows you to use a USB stick for storage and file transfer. Either native RomWBW formatted slices are supported, or FAT file systems using the **FAT.COM** utility. You need to be running RomWBW V3.4.0 or later though. (See below)

## RomWBW v3.4.0 is here!

  
Wayne Warthen has worked his magic and released a new major update of [RomWBW](<https://github.com/wwarthen/RomWBW/releases/tag/v3.4.0>). There are some great new features like support for front panel switches for boot options and also USB storage support (see above).  This now ships with RomBWB based RC2014 kits but you can burn your own rom using the **rcz80_std.rom** in the /Binary directory

## Tut-Tut Ported to RC2014

[Tut-Tut](<https://shieladixon.itch.io/tut-tut-port-for-rc2014-cpm>), the tomb raiding adventure game by David Stephenson has been ported to the RC2014 by Shiela Dixon.  This runs on any RC2014 from the Mini II to the Zed Pro, although you will need a TMS9918A video card, or a [TMSEMU](<https://www.tindie.com/products/shieladixon/tmsemu-graphics-card-designed-for-rc2014/>). A [sound card](<https://z80kits.com/shop/ym2149-sound-card/>) and [joystick module](<https://z80kits.com/shop/tynemouth-joystick-module-for-rc2014/>) add to the experience too.

## New Infocom Style Text Adventure

If you enjoy text adventure games, you'll love [The Ghosts of Blackwood Manor](<https://8bitgames.itch.io/ghosts>) which was released by Stefan Vogt. Built around the Infocom Z-Machine platform this dark and spooky game can be played on any RC2014 machine using the [Vezza](<https://gitlab.com/sijnstra1/vezza>) interpreter. (type **vezza-rw story.dat** to play)
