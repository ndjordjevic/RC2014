# Compact Flash Module

As many of you picked up on from the teaser picture last month, there is now a [Compact Flash Module](https://www.tindie.com/products/Semachthemonkey/compact-flash-module-for-cpm-rc2014-computer/) added to the RC2014 lineup!  
  
This works with the [Pageable ROM](https://www.tindie.com/products/Semachthemonkey/pageable-rom-module-for-rc2014-homebrew-computer/) and [64K RAM Modules](https://www.tindie.com/products/Semachthemonkey/64k-ram-module-for-rc2014-z80-homebrew-computer/) to allow the CP/M operating system to run on the RC2014!  
  
Although most of the components are through-hole items, and easy to solder, the CF socket itself can be fiddly if you're not used to surface mount soldering.  Therefore this module can also be purchased with the CF socket pre-mounted - although this may extend delivery times a little bit.  
  
The first batch of boards sold out really quickly, but new boards are due back in on Tuesday - however, to allow you to take advantage of the Easter Egg discount, I will be adding them back to the Tindie inventory on Sunday.
