## FORTH Part Of The Newsletter

If you're the kind of person that likes Reverse Polish Notation on your calculator, you'll be please to know that RC2014 user Rachel has ported [FORTH](https://en.wikipedia.org/wiki/Forth_\(programming_language\)) to the RC2014.  Her code is available on [GitHub](https://github.com/rachel8973/RC2014-FORTH) and can either be burned to a ROM, or loaded in to RAM

## Monitor ROM

If FORTH isn't close enough to the machine code for you, then user Steve has written a Machine Code Monitor for the RC2014.  It includes an assembler and disassembler, and everything else you'd expect. [Full details on his website](https://smallcomputercentral.wordpress.com/2017/11/18/monitor-v0-2-1/).

# RC2014 Goes Wireless

Those that followed along with the [Retro Challenge](http://www.retrochallenge.org/) this year would have seen me live streaming [Zork from the RC2014 ](https://rc2014.co.uk/1617/retro-challenge-201710-getting-started/)to the internet via an ESP8266 Wifi Module.  I may not have done great with Zork, but the prototype Wifi Module worked well.   
![](RC2014%20Newsletter%20-%20Issue%2010_files/d821e397-2da9-4bf7-8bad-a3f91bd06aae.jpg)  
---  
This newsletter should have been the launch of a Wifi Module based on the prototype. However, due to a schoolboy error, the PCBs will have to be redesigned, but should be available before Christmas.  
  
The schoolboy has been fed to a Grue
