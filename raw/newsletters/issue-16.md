## Zed and Zed Pro

Since the release of the 512k ROM 512k RAM Module, people have been asking for a complete kit running [RomWBW](https://www.tindie.com/products/Semachthemonkey/512k-rom-512k-ram-romwbw-module-for-rc2014/). Well, here it is, the [RC2014 Zed and RC2014 Zed Pro](https://www.tindie.com/products/Semachthemonkey/rc2014-zed-homebrew-z80-computer-kit/).  Available with a choice of compact flash or IDE hard drive, the Zed and Zed Pro also support Ed Brindleys RTC module as well as all your favourite RC2014 modules like the Digital I/O Module and Pi Serial Terminal.  
The Zed, did, of course, get it's name from the advanced CP/M software supplied on RomWBW, [Z-System](https://github.com/wwarthen/RomWBW/blob/master/Doc/ZSDOS%20Manual.pdf) (pronounced Zee System)

## Bus Raider!

For some users, the lack of memory mapped graphics on the RC2014 is an issue.  [Rob Dobson](https://groups.google.com/d/msg/rc2014-z80/ISZZeZuYj3w/doPggNHeAgAJ) saw this as a challenge though, and expanded on the Pi Serial Terminal idea with the [Bus Raider](https://robdobson.com/2018/09/raid-the-bus-and-the-speccy-lives/) Module. Not only does this give bitmapped graphics, but allows ZX Spectrum and TRS80 ROMs to run original software!
