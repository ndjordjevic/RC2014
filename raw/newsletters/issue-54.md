## Front Panel switches and LEDs for RomWBW

Since v3.4.0 RomWBW had had support for a simple front panel of switches and LEDs. This provides diagnostic info, disk access indication and allows you to set the boot up procedure of RomWBW.   
|  |  ![](https://mcusercontent.com/9794c5b78f76285b2bc1ff7bd/images/513e8a49-74e7-6612-5673-6b2288c5c328.jpg)  
---  
The [Universal Front Panel I/O Kit](<https://z80kits.com/shop/universal-front-panel-switch-and-led-for-romwbw/>) can be fitted to any enclosure, but next month will see the launch of the kit for the popular Blue Box enclosures. This will also take the [20x4 LCD module](<https://z80kits.com/shop/lcd-driver-module/>) launched last month.

## MSX8 - MSX Games on an RC2014

MSX8 is a CP/M program that Les Bird has been developing. It loads a customised MSX BIOS that works on an RC2014 with a TMS9918 or compatible video card. MSX games can then be loaded and run. Check the [GitHub](<https://github.com/lesbird/MSX8/tree/main>) page for code.
