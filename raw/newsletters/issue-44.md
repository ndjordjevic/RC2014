## New Product - NeoPixel Module from PeacockMedia

Not one but two new addressable RGB LED modules have been launched this week. The [NeoPixel Short Board](https://z80kits.com/shop/neopixel-module-short-board/) features 8 onboard LEDs and can drive up to 128 more pixels from a header on the side. The larger [8x8 Matrix board](https://z80kits.com/shop/neopixel-module-8x8-matrix/) is designed for the AdaFruit NeoMatrix and can also be extended with another 64 pixels.   
|  |  ![](RC2014%20Newsletter%20-%20Issue%2044_files/6d4122be-3d05-6f74-68be-84f09b6d9e4e.jpg)  
---  
Both modules are supplied fully assembled and include an SDK.  This allows easy programming in BASIC, C or Z80 assembly and include some pretty impressive demos.  
[Hackaday](https://hackaday.com/2023/01/19/addressable-leds-from-a-z80/) gave the original concept a nice writeup here  
  
Who's going to be the first to run a Super Star Trek scanner on one? ;-)

## Twitter to Mastodon Migration

  
Ever since Elon Musk took over Twitter, the days of RC2014 calling Twitter home were numbered. It is very sad to see what is happening to somewhere that was really very special for the RC2014 community. Whilst I haven't abandoned Twitter entirely, it is much lower down the priority list. Instead, you can find RC2014 on Mastodon on the oldbytes.space instance <https://oldbytes.space/@rc2014>  if you have jumped ship too.
