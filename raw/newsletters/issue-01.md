# New Pi Zero Module

The Pi Zero Module has been updated with extra features.  PCBs are at the fab house at the moment, and should be available on [Tindie](https://www.tindie.com/products/Semachthemonkey/raspberry-pi-zero-serial-terminal-for-rc2014/) some time next week.  
Add yourself to the waiting list to get notification when they're in!

# RC2014 Public Debut

Last month saw the public debut of RC2014 at Liverpool [MakeFest](https://lpoolmakefest.org/).  The event was a great success and attendees loved the RC2014 - particularly Filippos port of Snake to the platform.  
Read more about the event on [Google Groups](https://groups.google.com/d/msg/rc2014-z80/ctIFmhIbWsE/mdEhVwYBCAAJ)
